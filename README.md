# Piram-Drosera-Trap        

## О проекте         
  
Piram-Drosera-Trap — это ловушка на Solidity для проекта Drosera.  
Она отслеживает баланс токена PIRAM на определённом адресе и срабатывает, если баланс уменьшается более чем на 1 токен.

---

## Логика ловушки

- Отслеживается баланс токена PIRAM (ERC-20) по адресу:  
  `0x4a61B89655A5903224A1b82Eec50c320DD899e1c`

- Если баланс уменьшается более чем на 1 PIRAM (1e6 при 6 десятичных знаках), ловушка срабатывает.

- Контракт ловушки реализует обязательные для Drosera функции:
  - `collect()` — собирает текущий баланс.
  - `shouldRespond()` — анализирует данные и решает, сработать ли.

## drosera.toml

Конфигурационный файл Drosera содержит параметры подключения и настройки ловушки:

- `ethereum_rpc` — RPC Ethereum узла  
- `drosera_rpc` — RPC сервиса Drosera  
- `drosera_address` — адрес Drosera контракта  
- Раздел `[traps.piramtrap]` с параметрами ловушки: путь к ABI, контракт для реакции, функция для вызова, настройки cooldown, операторы и whitelist.
